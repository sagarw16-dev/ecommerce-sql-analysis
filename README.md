# E-Commerce Sales & Customer Analysis (SQL)

## Project Overview
This project analyzes e-commerce sales data using SQL to uncover insights related to revenue trends, customer behavior, and order performance.

## Tools Used
- SQL
- SQLite
- DB Browser for SQLite

## Key Analysis Performed
- Total and monthly revenue analysis
- Top customers by total spending
- Customer ranking using window functions
- Average delivery time calculation
- Repeat customer analysis
- Order status distribution

## Dataset
Brazilian E-Commerce Dataset (Olist) ‚Äì Kaggle

## Key Insights
- Revenue shows a consistent monthly growth trend
- A small group of customers contributes significantly to total revenue
- Average delivery time provides insights into logistics performance
